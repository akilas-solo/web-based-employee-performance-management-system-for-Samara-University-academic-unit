Troubleshooting

If you're still having issues logging in:

  1. Make sure your database server is running
  2. Verify that the database samara_evaluation exists
  3. Check that all tables have been created properly
  4. Ensure that the admin user exists in the users table
  5. Make sure the password is correctly hashed in the database
